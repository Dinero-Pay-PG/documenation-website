# Getting started

Dineropay gatway offers a complete, easy way to integrate your <u>_E-commerce_</u>.<br></br>
With our payment gateway you can accept all kind of payment method including _`applepay,stcpay`_ with a simple integration , our APIs designed in a RESTful like approch.
<br></br>
<br></br>

:::note

1. Registeration is free and it will give you access to a demo account<br></br>
2. You will find all the necessary keys to start the integration process

:::

## Making your first payment as easy as following the below steps
> 1. Make sure to [register](https://gate.dineropay.com/login?continue=%2F) in order to have the API key <br></br> 
> 2. Create a <u>`Brand`</u> in the dashboard <br></br>
> 3. Use one of our Ready to Go or using the prebuilt,custom integration <br></br>
