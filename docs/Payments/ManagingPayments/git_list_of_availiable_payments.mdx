# Available payment methods

Send this request providing, at the very least, the `brand_id` and `currency` query parameters having the same values you'd use to create your Purchase. Be sure to use the same API key you'll create your Purchase with

---

## Sample response
In order to get the available payment method You will need to make a <u>[`GET/ payment_methods`](../../../api/gateway/payment-methods)</u><br></br>

The `GET` request should be something like `/payment_methods/?brand_id=75a76529-91c7-4d98-90a9-8a641d70ee52&currency=SAR`
