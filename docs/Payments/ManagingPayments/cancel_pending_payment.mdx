# Cancel a pending payment

If you have a Purchase that payment is possible for, using this request you can guarantee that it won't be paid.

---

<br></br>

> - Nothing passed in the body for this operation <br></br>
> - You will have to make <u>[`POST /cancel`](../../../api/gateway/purchases-cancel)</u> request
> - You will pass the id of the transaction part as a path parameter

If the transaction was canceled successfully, in the response the status should be `cancelled`
